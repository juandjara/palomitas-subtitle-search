Subtitler-API Help

Subtitler-API is an API for searching in OpenSubtitles made by fuken using node-subtitler by divhide.

You can search subtitles for series with /search. You can indicate a text string for querying in the database, an imdb id, an episode number, a season number, a limit to the number of results that are displayed and a language for the subtitles. Default language is spanish. See this wikipedia article for more info about the language codes.
If a imdbid is indicated, query is ignored. If a query is indicated, imdbid is ignored (imdb_id has higher preference).

Examples: 

/search?imdbid=111111&season=3&episode=9&limit=1
/search?query=game of thrones&season=1&episode=1&limit=1&lang=eng
